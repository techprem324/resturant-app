# Site Map: Lumina Platform

The Lumina Restaurant Operating System structure:

```
├── Public Website
│   ├── Hero Section (Reservation, Order, Menu CTAs)
│   ├── Mood dial selector (Vibe filter)
│   ├── Menu Collection (Responsive grid, customizable items)
│   ├── Ingredient sourcing story (Interactive timeline)
│   ├── Client Booking & Interactive Seating Map
│   └── Visual photo gallery & Social activity feed
│
├── Guest Dashboard
│   ├── Auth status
│   ├── History (Reservations, orders)
│   ├── CRM taste tokens (Allergens, wine favorites)
│   └── Loyalty & points summary
│
├── Virtual Sommelier Chat Drawer
│   └── Wine cellar recommendations & active cart matching
│
└── Admin Operations Dashboard
    ├── CRM Guest Lookup
    ├── Table Layout & Live Waitlist auto-assignment
    ├── Demand Forecasting charts
    └── Supply chain status & Truffle auto-reordering
```

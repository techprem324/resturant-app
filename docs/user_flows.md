# User Flows: Lumina OS

Detailed workflows for critical user pathways.

## 1. Table Reservation Flow
```
[Visitor enters site] 
         │
         ▼
[Fills date/time/guests widget] 
         │
         ▼
[Opens interactive seating layout] 
         │
         ▼
[Selects specific table seat] 
         │
         ▼
[Submits form -> SMS OTP validation]
         │
         ▼
[Confirmation screen with QR code + automatic email confirmation]
```

## 2. Dynamic Ordering Flow
```
[Visitor goes to Menu]
         │
         ▼
[Applies vibe dial (e.g. Seductive)]
         │
         ▼
[Views mood-filtered menu cards]
         │
         ▼
[Selects item -> customize allergens/modifiers]
         │
         ▼
[Launches Sommelier for vintage wine recommendations]
         │
         ▼
[Adds wine to cart -> proceed to payment drawer]
         │
         ▼
[Stripe/UPI checkout transaction -> Success]
         │
         ▼
[Kitchen gets ticket -> Live order tracker status]
```

## 3. Admin Waitlist Automation
```
[Table 4 is cleared by floor staff]
         │
         ▼
[System alerts "Table 4 Open"]
         │
         ▼
[Admin clicks "Auto-Assign Tables"]
         │
         ▼
[AI selects top waitlisted party matching table capacity]
         │
         ▼
[Table assigned -> automated SMS sent to guest: "Your table is ready!"]
```
